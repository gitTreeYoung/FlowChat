FlowChat 隐私政策

最后更新：2026 年 4 月 4 日

概述

FlowChat（以下简称「本扩展」）是一款 Chrome 浏览器扩展，让用户在单个浏览器标签页中同时与多个 AI 平台对话。本隐私政策说明本扩展访问哪些数据、如何使用，以及哪些数据绝对不会被收集。

1. 我们不收集的数据

FlowChat 遵循严格的零收集原则。本扩展不会：

  • 收集、存储或传输您的 AI 对话内容或 Prompt 文本。
  • 记录您的浏览历史、访问的 URL 或扩展本身以外的标签页活动。
  • 收集任何个人身份信息（姓名、邮箱、IP 地址、设备标识符等）。
  • 向任何外部服务器、API 或第三方分析服务发送任何数据。
  • 使用 Cookie、追踪像素、浏览器指纹或任何形式的用户识别技术。
  • 在任何情况下与任何第三方共享数据。

2. 存储在您本地设备上的数据

本扩展仅通过 Chrome 内置的 chrome.storage API 将以下数据存储在您的本地设备上。这些数据永远不会离开您的设备。

2.1 同步设置（chrome.storage.sync）

随您的 Chrome 配置文件同步（与书签同步机制相同）：

键名内容
activePlatforms您已启用的 AI 平台实例列表
settings显示偏好：可见列数、布局模式、输入栏位置、窗口排序

2.2 本地设置（chrome.storage.local）

仅存储在当前设备：

键名内容
flowchat_selectors您为各 AI 平台输入框和发送按钮配置的自定义 CSS 选择器
flowchat_blocked_elements您选择在 AI 平台上隐藏的 UI 元素的 CSS 选择器
flowchat_groups您保存的会话组配置（平台列表及关联 URL）

所有本地存储的数据仅包含结构性配置信息，不包含消息文本、对话历史或 AI 生成的回复内容。

3. 本扩展如何访问第三方网站

FlowChat 将 AI 平台网站（ChatGPT、Claude、Gemini 等）以 iframe 形式嵌入扩展页面，以便您同时查看。本扩展：

  • 不拦截、读取、记录或存储您在这些平台上的对话内容。
  • 不修改 AI 回复内容，除非是您主动触发的操作（高亮标注、屏蔽特定 UI 元素）。
  • 向 AI 平台 iframe 注入轻量脚本（bridge.js、highlight.js），仅用于实现文字输入自动化和标注功能。
  • 对列出的 AI 平台移除特定 HTTP 安全响应头（X-Frame-Options、Content-Security-Policy），以使 iframe 嵌入在技术上成为可能。这是一项客户端修改，仅影响您的浏览器在本扩展内渲染这些页面的方式，不影响您直接访问这些网站时的安全性。

4. 权限及申请原因

权限申请原因
storage在本地保存您的扩展偏好设置
scripting向 AI 平台 iframe 注入输入自动化和标注脚本
tabs在扩展主页面和 AI 平台 iframe 之间路由消息
webNavigation检测 AI 平台 iframe 加载完成时机以注入脚本
declarativeNetRequest移除特定 AI 平台域名上阻止 iframe 嵌入的响应头
主机权限（19 个 AI 域名）允许向 AI 平台 iframe 注入脚本和传递消息

每项权限均仅用于其声明的用途。

5. 第三方服务

FlowChat 不集成任何第三方服务、分析平台、广告网络或数据经纪商。您通过本扩展使用的 AI 平台（ChatGPT、Claude 等）均受其各自隐私政策的约束，与本扩展无关，本扩展不代表您向这些平台共享任何数据。

6. 未成年人隐私

FlowChat 不故意收集 13 周岁以下儿童的任何信息。本扩展面向成年用户设计。

7. 政策变更

如本隐私政策有所更新，本文件顶部的「最后更新」日期将相应修改。重大变更将在扩展的版本更新说明中注明。

8. 联系方式

如您对本隐私政策有任何疑问，请通过 项目 GitHub 仓库提交 Issue，或通过 Chrome Web Store 商店页面联系开发者。
